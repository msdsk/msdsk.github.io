OnGeo

OnGeo is a long-lasting project to create an advanced solution for real property investors.

An innovative platform for generating reports about real estate, including all the information important for investors. It's built on knockout.js and it was the first js-framework project built in the company.

It includes a powerful and innovative, map-oriented real estate platform.

I was tasked with the website design and front-end implementation including tech choice and research.

This is an ongoing project, started in 2015, with a minor redesign in 2017.
